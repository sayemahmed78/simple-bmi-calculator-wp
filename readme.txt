== Description ==
Simple BMI Calculator WP is a lightweight WordPress plugin that allows users to calculate their Body Mass Index (BMI). This plugin provides an easy-to-use BMI calculator with customizable button color and border radius, making it simple to integrate into any WordPress site.

Plugin Documentation: [https://wordpress.org/plugins/simple-bmi-calculator-wp/](https://wordpress.org/plugins/simple-bmi-calculator-wp/)

= Usage =
* Use the shortcode `[bmi_calculator_wp]` to display the BMI calculator on any page or post.
* Go to "BMI Calculator WP" in the WordPress dashboard to customize the button color and border radius.

== Installation ==

= Minimum Requirements =

* WordPress 5.8 or greater
* PHP version 7.2 or greater

= AUTOMATIC INSTALLATION =
1. Log in to your WordPress dashboard.
2. Navigate to the Plugins menu and click Add New.
3. In the search field, type "Simple BMI Calculator WP" and click Search Plugins.
4. Once found, click Install Now.

= MANUAL INSTALLATION =
1. Download the plugin from [https://wordpress.org/plugins/simple-bmi-calculator-wp/](https://wordpress.org/plugins/simple-bmi-calculator-wp/).
2. Upload `simple-bmi-calculator-wp` to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.

= UPDATING =
Automatic updates should work seamlessly. Always back up your website before any update to avoid unforeseen problems.
